# 🚀 ChannelScope開発チャット（進捗管理付き）

このチャットは、SNSチャンネル分析アプリ **ChannelScope** の開発専用スレッドです。  
目的は、YouTube・TikTokなどのデータを自動収集し、ダッシュボードで可視化・分析すること。  
ChatGPTとCursor（またはVSCode）を連携して継続的に開発を進めます。

---

## 🧠 現在のプロジェクト情報

| 項目 | 内容 |
|------|------|
| **アプリ名** | ChannelScope |
| **目的** | SNSチャンネルの成長データ（再生数・いいね・登録者など）の可視化 |
| **開発環境** | PHP 8 / MySQL 8 / XAMPP（ローカル） |
| **DB名** | `purplelion51_channel_scope_db` |
| **サーバー** | さくらのレンタルサーバー（本番予定） |
| **API連携** | YouTube Data API v3（APIキー名：`sorciere-youtube-api`） |
| **Composer** | インストール済（グローバル利用可） |

## 📊 今後の開発予定（優先度順）

| 優先度 | 項目 | 内容 |
|:--:|:--|:--|
| ⭐ | **グラフ機能追加** | Chart.jsで再生数・いいね数の推移を折れ線グラフ化 |
| ⭐ | **期間フィルタ** | 「直近7日」「30日」「全期間」で動画集計を切り替え |
| ⭐ | **TikTok API連携** | マルチプラットフォーム化（同DB内で共存） |
| ★ | **自動更新処理** | cronで1日1回 `youtube_import.php` 実行 |
| ★ | **UIデザイン整備** | TailwindまたはBootstrap導入で統一感を出す |
| ☆ | **本番デプロイ** | さくらサーバーへのDB・コード配置、APIキー保護 |
