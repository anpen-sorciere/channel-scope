## 1. プロジェクト概要

YouTube と TikTok の運営アカウントを対象に、  
**「なぜ伸びないのかを可視化し」「どう改善すべきかを提案する」**ことを目的とした、  
クリエイター向け成長支援 Web アプリケーションを開発する。

### 1.1. 想定ユーザー
- YouTube / TikTok に 1〜10万人規模のフォロワーを持つクリエイター  
- 自分のチャンネルを“感覚ではなくデータで改善したい”人  
- 投稿ジャンル・時間・タグを科学的に調整したいユーザー

### 1.2. 対応プラットフォーム
- YouTube  
- TikTok  
（将来的に Instagram・X など追加可能）

---

## 2. 目的・ゴール

### 2.1. ビジネス目的
- フォロワー伸び率を改善し、投稿効率を最大化する。  
- 将来的には SaaS 化（他のクリエイターにも提供）を見据える。

### 2.2. ユーザー価値
- 「どの時間帯・テーマが伸びているか」が可視化できる。  
- AIによる改善提案で、次の投稿戦略を立てやすくする。  
- 投稿効果をレポート形式で振り返りできる。

---

## 3. 機能一覧

### 3.1. 分析機能
- チャンネル全体統計（フォロワー数、再生数、エンゲージメント）
- 投稿時間帯分析（曜日×時間ヒートマップ）
- タグ・ジャンル分析（キーワードごとの平均パフォーマンス）
- トレンド分析（伸び率上昇／下降テーマの検知）

### 3.2. 改善提案機能
- 投稿時間帯の最適化（おすすめ時間帯自動抽出）
- タグ／シリーズ別のパフォーマンス比較
- AIコメントによる改善ヒント出力  
  例：「TikTokでは短尺ASMRが好調、平日18〜21時投稿を推奨」

### 3.3. レポート機能
- 週次・月次レポート（伸び率・TOP動画・改善項目）
- グラフによる推移可視化（再生数・フォロワー数・時間別分布）

### 3.4. 管理・設定
- アカウント連携（YouTube OAuth / TikTok Business API）
- データ期間の切替（30日／90日／1年）
- 通知・レポート配信設定（将来拡張）

## 4. 画面仕様

### 4.1. ダッシュボード
- 再生数・登録者・いいね率などのサマリーカード
- 直近推移グラフ
- 「伸びた動画TOP5」「落ちた動画TOP5」テーブル

### 4.2. 時間帯分析
- ヒートマップ表示（曜日×時間×再生数）
- AIコメントによる推奨時間帯アドバイス

### 4.3. タグ・キーワード分析
- タグ出現頻度×平均再生数を集計
- 上昇・下降トレンドを色分け表示

### 4.4. 動画詳細一覧
- 各動画の投稿時刻・再生数・いいね率・コメント率
- プラットフォームごとにソート・絞り込み

## 7. 分析アルゴリズム概要

### 投稿時間帯スコア
score = AVG(views_per_video_in_slot) / AVG(views_all)
→ スコア > 1.2 → 推奨時間帯

### タグ分析
- タグごとに平均再生数を算出  
- 全体平均と比較して ±20%以上を色分け

### トレンド検知
recent_avg < past_avg * 0.8 → ネガティブトレンド
